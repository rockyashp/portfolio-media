# Yash Photography Portfolio

A sleek, high‑performance web showcase for **Yash Patil**, a contemporary photographer based in 2026.  
Built with **Vite + React**, styled with vanilla CSS, and featuring:

- **Responsive gallery** with smooth transitions and arrow navigation.  
- **Dynamic inquiry form** with date validation and error handling.  
- Optimised image assets (SVG logo cropping, compressed media).  
- Dark‑mode aesthetic, subtle animations, and glass‑morphism touches.  

## Features

- **Image & video gallery** – thumbnails and full‑screen lightbox.  
- **Inquiry booking slip** – send print/photography requests, receive confirmation.  
- **Print‑ready SVG logos** – Lightroom, Premiere, DaVinci Resolve (cropped for crisp display).  
- **Performance‑focused** – Vite build ~350 KB gzip, lazy‑loaded assets.
